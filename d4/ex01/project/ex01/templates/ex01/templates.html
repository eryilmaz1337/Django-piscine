Django Template Engine

Django şablon motorunun temel özellikleri:

  • Blocks: {% block name %} ... {% endblock %} ile kalıtım yapılır.
  • Loops: {% for x in y %} ... {% endfor %} ile döngü kurulur.
  • If: {% if condition %} ... {% endif %} ile mantıksal kontrol yapılır.
  • Variables: {{ variable }} ile veriler ekrana basılır.
